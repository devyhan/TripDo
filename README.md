# TripDo
코로나때문에 가기 힘들어진 여행의 계획을 미리 짜보고 미리 루트를 확인할 수 있는 여행 투두앱 입니다.

## Architecture

- MVC(리팩터링 필요)

## Requirements
- **Language**
  - Swift 5.0
- **Framework**
  - UIKit
  - CoreData
  - CocoaPods
  - SPM(SwiftPackageManager)
- **IDE**
  - Xcode
